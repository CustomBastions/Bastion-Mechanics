This is a list of all terms that may require definitions.

# Bastion Terms

| Term             | Definition                                                                                                                                                                                           |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Bastion          | The bastion is the player base itself. It is what this whole system revolves around.                                                                                                                 |
| Communal Bastion | A communal bastion is a type of bastion that is managed by multiple player characters. You can find the changes to the rules in [[Communal Bastions]].                                               |
| Joined Bastion   | A joined bastion is the term for multiple solo bastions being placed together, such that they can function as a single party base, but where each player character still controls their own bastion. |
| Solo Bastion     | A solo bastion is a bastion that's run by a single player character. It is generally assumed that it is the bastion type talked about unless specified.                                              |

# Square Terms

| Term                        | Definition                                                                                                                                                                                                                                                                                                                                      |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Bastion Squares             | These are squares that have been added to the bastion. They can contain a single room square.                                                                                                                                                                                                                                                   |
| Built Squares               | These are the squares that have been built, be it for the bastion as a whole or individual rooms. <br><br>For bastion squares they express the total amount of squares available for making room squares. <br><br>For room squares they express the total amount of squares available for the room's core feature and additional room features. |
| Free [Bastion/Room] Squares | These are squares that have been built, but either don't have a room or room feature yet. The former is for bastion squares while the latter is for room squares.                                                                                                                                                                               |
| Max Squares                 | Max squares are typically used for rooms. They designate the max number of built squares the room can have. More room squares can be built beyond this limit, but they do not count towards "built squares" and are purely decorative.                                                                                                          |
| Room Squares                | Room squares are bastion squares that a room has been built or expanded into. Each room has their own room squares.                                                                                                                                                                                                                             |
| Square                      | A square when referenced as a unit is a 5ft. x 5ft. square. It can also be treated as a 1.5m x 1.5m square. This is the unit used for any and all measurements in building a bastion.                                                                                                                                                           |
| Used Squares                | Used squares are squares that already have a room or is used for the base room ability or added room features for bastion and room squares respectively.                                                                                                                                                                                        |
| Decorative Square           | These squares are room squares that go above the max squares value. These squares server no purpose except being decoration to allow the player to make their bastion look as they wish. They still cost the same as the room's regular square, but talk with your DM.                                                                          |


# Room Terms

| Term              | Definition                                                                                                                                                                                                                                                 |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Cost to Build     | The cost of building the structure. It is in GP unless otherwise specified.                                                                                                                                                                                |
| Cost to Expand    | The cost of converting a bastion square into a room square of that specific room.                                                                                                                                                                          |
| Exclusivity Group | An exclusivity group is a value that can be given to a structure. If multiple structures share the same exclusivity group, then a solo bastion can only ever have one of said structures at any given time.<br>They work differently in communal bastions. |
| Expansion Slots   | The number of expansion slots you use while building the room.                                                                                                                                                                                             |
| Hirelings         | The number of hirelings the room needs to run.                                                                                                                                                                                                             |
| Initial Size      | The number of connected free bastion squares needed to build the room. These squares will be count as used squares.                                                                                                                                        |
| Max Size          | This will become the value for the room's max squares.                                                                                                                                                                                                     |
| Minimum Level     | The minimum level the character needs to be to build the structure.                                                                                                                                                                                        |
| Requirements      | Any kind of requirement that needs to be fulfilled to build the structure. It can be a specific arrangement of squares, a specific class or species feature needed, or anything else.                                                                      |
| Room              | The main way bastions affect the game.                                                                                                                                                                                                                     |
| [Room] Feature    | Room features are expansions to room that add new effects or improve the existing ones. They may sometimes just be called "features", especially when mentioned together with rooms.                                                                       |
| Structure         | Structure is the general term to refer to both rooms and room features.                                                                                                                                                                                    |
| Time to Build     | The number of bastion turns it takes to build the structure.                                                                                                                                                                                               |
|                   |                                                                                                                                                                                                                                                            |

# General Rule Terms

| Term            | Definition                                                                                                                                                                                            |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Actions         | Every room can take one action per bastion turn. They are stronger effects.                                                                                                                           |
| Bastion Turn    | A bastion turn is the measurement of time between stuff happens at the bastion. It is generally 7 days, but the DM can change it as needed.                                                           |
| Expansion Slot  | Expansion slots are used when building new squares or structures. You have a limited amount available at any given time. They are only unavailable so long as they are being used to build something. |
| Hireling        | Hirelings are people that come and work in your rooms. They are generally automatically hired and the rooms generate enough to pay for them. More specifics can be found under the hireling rules.    |
| Housed Hireling | A type of hireling that needs a place to live in the bastion.                                                                                                                                         |
| Perks           | Passive effects that a structure provides. This can be all new effects or enhance existing perks or actions, typically of their room, but it can affect others.                                       |
